# Ohjelmistoprojekti

## Asennus

Projekti asennettiin BASH-komennoilla: 

`npx create-react-app . --template typescript`
Loi React-sovelluksen TypesScript:lla [https://create-react-app.dev/docs/getting-started/#creating-a-typescript-app](https://create-react-app.dev/docs/getting-started/#creating-a-typescript-app).

`npm install`
Asennetaan kaikki sovelluksen riippuvuudet.

## Projektin käynnistäminen

Täysin toimiakseen, projektiin tarvitaan myös sen backend, joka löytyy seuraavasta linkistä [https://gitlab.tamk.cloud/JanneSaukkio/ostoslista-backend](https://gitlab.tamk.cloud/JanneSaukkio/ostoslista-backend).

Seuraa backendin asennusohjeita.

Kun backend on asennettu sekä se on ajossa, voimme käynnistää frontendin.

Käynnistäminen tapahtuu komennolla `npm start`.
